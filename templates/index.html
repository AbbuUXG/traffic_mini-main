Traffic Sign Recognition System

Real-time recognition of traffic signs using computer vision and deep learning

About

The Traffic Sign Recognition System is an application that uses computer vision and deep learning techniques to identify traffic signs in real time. It can detect various traffic signs such as speed limits, stop signs, yield signs, and more. The system analyzes the input from a camera feed and provides information about the detected signs, including the class and probability.
